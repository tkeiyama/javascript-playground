## Package Names

### `@javascript-playground/{core}-{purpose}`

ex. `@javascript-playground/react-vite`

- `{core}` is a core tech.
  - ex. `typescript`, `react`
- `{purpose}` is what the package does.
  - ex. `vite`, `custom-hook`
